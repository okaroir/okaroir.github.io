General Description

  • Checkboxes with custom colors are designed based on the colors provided in the template.
  • To learn about the colors of the template, you can refer to the colors section.
  • To customize the checkboxes, refer to the assets/css/scss/components/forms/checkboxes/_checkboxes.scss address.
  • To customize the checkboxes in dark mode, refer to the assets/css/scss/components/forms/checkboxes/_checkboxes_dark.scss address.
  • To create a checkbox, you must first add .form-check-input class to the element.
  • Add additional classes to the desired element (the element that has the form-check-input class) based on different styles and colors (which can be seen in full below).

Colors

  • We use 9 color schemes in the template.
  • Exclusive checkboxes are designed based on these 9 colors.
Red Blue Cyan Orange Purple Green Pink Yellow Gray

Color Weights

  • We have used 10 weights for coloring, which are:
    dark-4 - dark-3 - dark-2 - dark-1 - medium - light-1 - light-2 - light-3 - light-4 - light-5
  • To use different color weights, just apply .checkbox-{color}-{weight} class to the desired element. For example: .checkbox-red-dark-4

Background Colors

Using the .checked-bg-{color}-{weight} class, you can change the background color of the checkbox when it is checked . For example: .checked-bg-gray-dark-1
Using the .unchecked-bg-{color}-{weight} class, you can change the background color of the checkbox when it is unchecked . For example: .unchecked-bg-red-light-5

Border Color

Using the .checked-border-{color}-{weight} class, you can change the border color of the checkbox when it is checked . For example: .checked-border-black
Using the .unchecked-border-{color}-{weight} class, you can change the border color of the checkbox when it is unchecked . For example: .unchecked-border-red-dark-4

Using the .checked-mark-{mark} class, you can change the style of the mark when the checkbox is checked . For example: .checked-mark-square-01
Using the .checked-mark-{color}-{weight} class, you can change the color of the mark when the checkbox is checked . For example: .checked-mark-cyan-medium
Using the .unchecked-mark-{mark} class, you can change the style of the mark when the checkbox is unchecked . For example: .unchecked-mark-square-01
Using the .unchecked-mark-{color}-{weight} class, you can change the color of the mark when the checkbox is unchecked . For example: .unchecked-mark-cyan-medium

Sizes

To change the size of the checkboxes, you can use custom classes including: .checkbox-xs ، .checkbox-sm ، .checkbox-md ، .checkbox-lg و یا .checkbox-xl
In this case, the size of the marks changes based on the size of the checkbox.
In addition, you can use the width and height classes to change the size of the checkboxes.
In this case, the size of the signs does not change based on the size of the checkbox, and you have to manually determine the font size of the mark.

Pulsating checkboxes

If you add the .checked-pulse class to the checkbox, the checkbox will pulse when it is checked
If you add the .checkbox-unchecked-pulse class to the checkbox, the checkbox will pulse when it is unchecked
The pulses you can apply are: scale-01، scale-02، scale-03، scale-04 و scale-05
To apply new pulses, just add the .pulse-{pulse} class to the element. for example: .pulse-scale-03
For change the animation duration, you can use .anim-duration-{duration} class. for example: .anim-duration-200ms
For more information, you can refer to the animation duration section.
For add the animation delay, you can use .anim-delay-{delay} class. for example: .anim-delay-1s
For more information, you can refer to the animation delay section.
For change the animation iteration count, you can use .anim-count-{count} class. for example: .anim-count-1
For more information, you can refer to the animation iteration count section.
