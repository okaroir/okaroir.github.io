General Description

  • We have customized the bootstrap buttons based on the colors provided in the template.
  • In addition to Bootstrap's default buttons, custom and exclusive buttons are also included in the template.
  • To learn about the colors of the template, you can refer to the colors section.
  • To customize the buttons, refer to the assets/css/scss/components/forms/buttons/_buttons.scss
  • To customize the buttons in dark mode, refer to the assets/css/scss/components/forms/buttons/_buttons_dark.scss
  • To create a button, you must first add the .btn to the elements.
  • Add additional classes to the element (the element that has the .btn class) based on different styles and colors (which can be seen in full below).

Colors

  • We use the following 9 color schemes in the template.
  • Exclusive buttons are designed based on these 9 colors.
  • Bootstrap buttons are customized based on these 9 colors.
Red Blue Cyan Orange Purple Green Pink Yellow Gray

Colors Weights

  • We have used 10 weights for colors, which are:
    dark-4 - dark-3 - dark-2 - dark-1 - medium - light-1 - light-2 - light-3 - light-4 - light-5
  • To use different color weights, just add the .btn-{color}-{weight} class to the element. For example: .btn-blue-medium

Sizes

  • By default, Bootstrap has considered only two sizes for buttons, small and large, but we have considered all sizes, which are:
    Extra Small, Small, Medium, Large, and Extra Large
  • To change the size of the buttons, just apply one of the following classes to the elements:
    • btn-xs
    • btn-sm
    • btn-md
    • btn-lg
    • btn-xl

Simple Buttons

  • These buttons have a background color.
  • Just apply the .btn-{color}-{weight} class to the element. For example: .btn-red-dark-4

Outline Buttons

  • These buttons have no background color, but have a colored border and text.
  • When the mouse is hovered on each button, the background color and the text color change.
  • Just apply the .btn-outline-{color}-{weight} class to the element. For example: .btn-outline-blue-medium
  • To style borders, you can use .border-dotted ، .border-dashed ، .border-solid و یا .border-double classes.
  • To change the border size, you can use the .border-{size} class.
